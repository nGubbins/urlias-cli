# URL IA & S
## URLs, Information Architecture, & Scraping

# ALL COMMANDS
## S - set url
  Set the URL to be used by other commands. If you run a command without a URL set you will prompted to add one. Currently, please include the URI scheme such as 'https://'

## C - check url status
  Returns the status of the URL (200, 404, 403, etc)

## P - ping url
    Single response time / latency check.
